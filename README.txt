This plugin integrates a Vertical Booking reservation form.

== Description ==

Integrate Vertical Booking reservation painlessly in your website.

The plugin is developed using AngularJs, Moment.js and Lodash.
JavaScript must be activated for the plugin to work.

== Installation ==

As simple as it gets:

1. Upload the plugin
2. Activate it
3. Configure it by setting the 4 option fields under Settings -> Visa Vertical Booking
4. Place [vvb_display_form] shortcode wherever you want

== Frequently Asked Questions ==

= Where do I get the values for the settings? =

Check Vertical Booking contract
